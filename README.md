# grid-research

Vue3 에서 사용할 그리드 컴포넌트 적용 & 판단하기

## 대상 컴포넌트

* **Tabulator**
  * https://tabulator.info/
  * npm install tabulator-tables --save
  * https://tabulator.info/docs/6.3

* **Primevue**
  * https://primevue.org/
  * npm install primevue @primevue/themes
  * https://primevue.org/vite/

## 필수 확인사항

* 컬럼 길이 조정
* 컬럼 순서 변경
* 멀티 정렬
* 개인 설정정보 저장
* 커스텀 테마
* 컬럼 영역에 vue template 적용
